Evolution of Computing :
    dedicated servers :
        single application/enterprise
    virtual machines :
        hypervisor provides multiple vm's within same machine
        one application/enterprise uses one vm
    containers :
        multiple containers within a single vm
        better use of space
    functions :
        function executes a piece of code put on it
        single contains multiple functions

        sharing the cost with other customers
    stop guessing capacity :
        scale up or down
    increases agility and speed :
        launch resource within few clicks
    stop spending money on running and maintaining data centers
